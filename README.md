# Droid Sans Fallback

## Description

This is a fork of 'Droid Sans Fallback', which removes buggy `vert` feature from original, some erroneous or non-fullwidth characters have also been removed.

This font does not contain any Western characters (like Latin, Greek, and Cyrillic); over 99% of the characters are CJK characters (Hanzi, Bopomofo, Kana, and Hangul), with the remainder consisting of fullwidth Latin letters, characters from the 'CJK Compatibility' block, and some punctuation marks. Therefore, if you want to use it in documents and on websites, you have to specify a separate Western font to complement it.
